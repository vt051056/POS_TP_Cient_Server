   Инсталляция альтернативного  USB - драйвера.

 Для работы с USB напрямую (не виртуальный COM-порт) сначала надо установить драйвера.
 В начале стандартный драйвер STM виртуального порта - "VCP-drv.rar"    (https://yadi.sk/d/Q2SB9f8rcLs62)
 Затем – драйвер libusb win32 - "libusb-win32-bin-1.2.6.0.zip "   (https://yadi.sk/d/uyDWJWCRhjqU7)
  И теперь можно удалить стандартный драйвер STM ("VCP-drv.rar").
